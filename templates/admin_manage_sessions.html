    Session Management Guide
    How Sessions Work:
    • Each session maintains separate data (classes, enrollments, teacher assignments)
    • Only one session can be Current at a time
    • New data (students, classes, teachers) is added to the current session
    • You can view data from any historical session
    Best Practices:
    • Create a new session at the start of each academic year
    • Copy important data from previous session if needed
    • Mark the new session as "Current" to start adding data
    • Historical sessions remain accessible for reporting
